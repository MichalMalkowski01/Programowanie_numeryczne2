# C Programming Course

Materiały i przykłady z zajęć z języka C.

Repozytorium zawiera zadania, rozwiązania oraz kod omawiany podczas kursu.

## zakres kursu

* podstawy języka C (typy, pętle, warunki)
* operacje na liczbach i tablicach
* funkcje
* wskaźniki
* przetwarzanie danych
* podstawowe algorytmy (search, sort)

## struktura repozytorium

Każdy folder (`01`, `02`, `03`, ...) odpowiada kolejnym zajęciom.

W środku:

* pliki `.c` → rozwiązania i przykłady
* `ZadXX.pdf` → treść zadań
* `output/` → przykładowe wyniki (jeśli potrzebne)

## jak uruchomić

Każdy plik kompilowany osobno:

```bash
gcc nazwa_pliku.c -o app
./app
```

Brak zależności zewnętrznych — czysty C.

## założenia

* kod pisany od zera (bez gotowców)
* nacisk na czytelność i zrozumienie
* rozwiązania celowo proste (pod naukę, nie optymalizację)
* każdy temat rozwijany stopniowo między zajęciami

## przykładowe zadania

* znalezienie drugiej największej liczby
* sprawdzanie palindromu
* generowanie ciągu Fibonacciego
* operacje na tablicach (średnia, suma, duplikaty)
* odwracanie danych

## cel

Przygotowanie do:

* pisania kodu w C od podstaw
* rozumienia działania pamięci i wskaźników
* dalszej nauki struktur danych i algorytmów
